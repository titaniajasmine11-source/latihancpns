# Database Setup

Jalankan `schema.sql` terlebih dahulu di Supabase SQL Editor, lalu jalankan `seed.sql`.

File ini menyiapkan tabel utama, RLS dasar, trigger profil user baru, kategori TWK/TIU/TKP, topik awal, dan konfigurasi skor/generator.

Policy admin yang lebih lengkap akan ditambahkan saat halaman admin dan server action sudah tersedia.
